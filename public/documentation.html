My FlixApi

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Response Request body data format
Get a list of all movies /movies GET A JSON object holding data about all movies. None
Get specific movie by name /movies/:movieId GET A JSON object holding data about a specific movie. None
Get data about a genre by name. /genres/:genre GET A JSON object holding data about a specific genre. None
Get data about movie director /directors/:name GET A JSON object holding data about a movie director. None
Create new user /users/create POST A JSON object holding data about a user that was added. A JSON object holding data about the new user, structured like:
{
"firstName": "John",
"lastName": "Doe",
"age": 30,
}
Update specific user information /users/:userId PUT A JSON object holding data about a user that was updated. None
Delete user /users/:userId/delete DELETE A string that indicates whether the user was deleted. None
Add movie to favourites /users/:userId/:movieId/favorites POST A string confirming which movie was added successfully to favourites. A JSON object holding data about film, structured like:
{
title: "Avengers: Endgame",
year: "2019",
}
Delete movie from favourites /users/:userId/:movieId/favorites DELETE A string that indicates whether the movie was successfully removed from favourites. None
