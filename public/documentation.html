My FlixApi

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Response Request body data format
Get a list of all movies /movies GET A JSON object holding data about all movies. None
Get specific movie by name /movies/:Title GET A JSON object holding data about a specific movie. None
Get data about a genre by name. /genre/:Name GET A JSON object holding data about a specific genre. None
Get data about movie director /directors/:Name GET A JSON object holding data about a movie director. None
Create new user /users POST A JSON object holding data about a user that was added. A JSON object holding data about the new user, structured like:
{
"Name": "militao",
"Born": "1995/02/15",
"Email": "mirannn@gmail.com"
}
Login /login POST A JSON object holding data about a user who was logged in A JSON object holding data about the logged in user, structured like:
"user": {
"_id": "621e0cf3b9cc4f68235ea2be",
"Name": "Miki9",
"Born": "1995-02-02T00:00:00.000Z",
"Email": "mare@gmail.com",
"Password": "$2b$10$cycqj8dP.QbT7ry/CMFWNOuDSXpx109MHLdvEVeaIdWSDFpAuqaeu",
"FavoriteMovies": [],
"__v": 0
},
"token": ""
}
Update specific user information /users/:Name PUT A JSON object holding data about a user that was updated. A JSON object holding data used to update the user, structured like:
{ "Name": "Otto"
}
Delete user /users/:Name DELETE A string that indicates whether the user was deleted. None
Add movie to favourites /users/:Name/favorites/:MovieId POST A JSON object holding data about a user where movie was added A JSON object holding data about a user where movie was added,structured like:
{
"Name": "Mared5",
"Born": "1995-02-01T23:00:00.000Z",
"Email": "mare@gmail.com",
"Password": "1234",
"FavoriteMovies": [
"62162abdae70179e807a3eb2"]
}
Delete movie from favourites /users/:Name/favorites/:MovieId DELETE A JSON object holding data about a user where movie was deleted. A JSON object holding data about a user where movie was deleted,structured like:
{
"Name": "Mared5",
"Born": "1995-02-01T23:00:00.000Z",
"Email": "mare@gmail.com",
"Password": "1234",
"FavoriteMovies": []
}
